# 论文格式化工具使用指南

欢迎使用论文格式化工具！这个工具旨在帮助您快速、准确地将毕业论文格式调整为符合学校要求的标准格式。无论您是第一次使用这类工具，还是对复杂的Word格式设置感到头疼，本指南都将为您提供详尽的帮助。您只需在界面上简单设置参数，工具就能自动完成格式调整工作，让您有更多时间专注于论文内容。

工具界面主要分为左侧的七个功能选项卡和底部的操作区域。请按照本指南的说明，逐步设置每个选项卡中的参数。

---

## 选项卡 1: 页面设置

此选项卡用于设置论文的整体页面布局和边距。

- **上边距** (单位: cm, 默认值: 2.4)
  - 设置页面顶部到正文内容区域的距离。点击上下箭头或直接输入数字来调整。增大该值会使正文内容向下移动。
- **下边距** (单位: cm, 默认值: 2.4)
  - 设置页面底部到正文内容区域的距离。
- **左边距** (单位: cm, 默认值: 2.4)
  - 设置页面左侧到正文内容区域的距离。
- **右边距** (单位: cm, 默认值: 2.4)
  - 设置页面右侧到正文内容区域的距离。
- **装订线** (单位: cm, 默认值: 0.5)
  - 为论文装订预留的额外空间。此空间通常加在左边距，以确保装订后文字不被遮挡。
- **页眉距** (单位: cm, 默认值: 1.5)
  - 设置页面顶部到页眉内容的距离。
- **页脚距** (单位: cm, 默认值: 1.25)
  - 设置页面底部到页脚（通常是页码）内容的距离。

---

- **前置页处理** (下拉菜单, 默认值: "自动识别")
  - 控制工具如何处理论文正文前的页面（封面、声明、摘要等）。不同学校对这些页面的格式要求差异很大。
  - 选项：
    - **自动识别**：工具会尝试智能识别前置页并进行格式化。
    - **跳过（不处理）**：完全忽略前置页，不做任何格式修改。适用于您已手动设置好前置页的情况。
    - **强制格式化**：强制对前置页进行格式化。请谨慎使用，可能需要后续手动调整。

---

## 选项卡 2: 正文样式

此选项卡用于设置论文正文的字体、字号、对齐方式以及段落间距等。

- **西文字体** (输入框, 默认值: "Times New Roman")
  - 设置正文中所有英文字符和数字的字体。输入字体名称即可，例如 "Arial"、"Calibri" 等。
- **正文中文字体** (输入框, 默认值: "宋体")
  - 设置正文中所有中文字符的字体。例如 "微软雅黑"、"楷体" 等。
- **正文字号** (下拉菜单, 默认值: 12pt)
  - 设置正文文本的字号大小。可选：9pt / 10.5pt / 12pt / 14pt / 16pt / 18pt / 22pt / 24pt / 26pt / 36pt。
- **正文对齐** (下拉菜单, 默认值: "两端对齐")
  - 设置正文段落的对齐方式。
  - 选项：保持原样 / 左对齐 / 居中 / 右对齐 / 两端对齐。
- **首行缩进** (单位: pt, 默认值: 24)
  - 设置正文段落第一行文本的缩进量。12pt 相当于一个中文字符的宽度，24pt 即缩进两个汉字。
- **行距** (单位: 倍, 默认值: 1.5)
  - 设置正文段落的行间距离。通常设置为 1.25 倍或 1.5 倍。
- **段前** (单位: 行, 默认值: 0)
  - 设置段落上方与前一个段落之间的间距。
- **段后** (单位: 行, 默认值: 0)
  - 设置段落下方与后一个段落之间的间距。

---

- **脚注字号** (下拉菜单, 默认值: 9pt)
  - 设置脚注文本的字号大小。
- **脚注行距** (单位: 倍, 默认值: 1.0)
  - 设置脚注段落的行间距离。

---

## 选项卡 3: 标题样式与编号

此选项卡用于设置论文各级标题的样式和自动编号规则。

### 各级标题样式

工具为一级标题（H1）、二级标题（H2）、三级标题（H3）和四级标题（H4）提供了独立的样式设置区域。每个标题级别包含以下设置：

- **字体** (输入框) — 设置该级标题的字体。
- **字号** (下拉菜单) — 设置该级标题的字号大小。
- **加粗** (下拉菜单) — 控制是否加粗。选项："加粗"、"不加粗"、"保持原样"。
- **对齐** (下拉菜单) — 设置对齐方式。选项："保持原样"、"左对齐"、"居中"、"右对齐"、"两端对齐"。
- **段前 / 段后** (单位: 行) — 设置标题上下方的间距。输入 `-1` 表示不改变原有间距。

各级标题的默认值：

| 级别 | 字体 | 字号 | 加粗 | 对齐 | 段前/段后 |
|------|------|------|------|------|-----------|
| H1 一级标题 | 黑体 | 14pt | 加粗 | 左对齐 | 0 / 0 |
| H2 二级标题 | 黑体 | 12pt | 加粗 | 左对齐 | 0 / 0 |
| H3 三级标题 | 楷体 | 12pt | 不加粗 | 左对齐 | 0 / 0 |
| H4 四级标题 | 楷体 | 12pt | 不加粗 | 左对齐 | 0 / 0 |

### 标题编号设置

- **自动修正标题编号** (复选框, 默认: 选中)
  - 建议保持选中。工具会根据编号预设自动为各级标题生成或修正编号，确保编号连续正确。
- **编号预设** (下拉菜单, 默认: "第X章 / X.X / X.X.X (SCAU)")
  - 选择预设的标题编号格式，会自动填充下方的编号模式。
  - 选项：
    - **第X章 / X.X / X.X.X (SCAU)**：华南农业大学本科论文常用格式。
    - **X / X.X / X.X.X (纯数字)**：纯数字编号，如 "1"、"1.1"、"1.1.1"。
    - **一、/ (一) / 1. (中文序号)**：中文数字编号，如 "一、"、"(一)"、"1."。
    - **Chapter X / X.X / X.X.X (英文)**：英文论文常用格式。
- **一级标题** (输入框, 提示: 如 第1章) — 定义一级标题的编号模式。
- **二级标题** (输入框, 提示: 如 1.1) — 定义二级标题的编号模式。
- **三级标题** (输入框, 提示: 如 1.1.1) — 定义三级标题的编号模式。
- **四级标题** (输入框, 提示: 如 1.1.1.1) — 定义四级标题的编号模式。

## 选项卡 4: 页眉与页码

此选项卡用于设置论文的页眉内容、显示规则以及页码的格式和位置。

### 页眉

- **启用页眉** (复选框, 默认: 未选中)
  - 控制是否在论文中应用页眉。选中后下方所有页眉设置才会生效。
- **作用范围** (下拉菜单, 默认: "仅正文")
  - 选项："仅正文"（前置页不显示页眉）、"全部"（所有页面都显示）。
- **奇偶页不同** (复选框, 默认: 选中)
  - 控制奇数页和偶数页的页眉内容是否可以不同。
- **首页不显示页眉** (复选框, 默认: 未选中)
  - 控制每章的第一页是否显示页眉。
- **奇数页(右)** (输入框, 默认: "华南农业大学 硕士学位论文")
  - 设置奇数页页眉的内容。
  - **↳ 自动显示章标题** (复选框) — 选中后页眉将自动显示当前章节标题。
- **奇数页对齐** (下拉菜单) — 设置奇数页页眉的对齐方式。
- **偶数页(左)** (输入框) — 设置偶数页页眉的内容。
  - **↳ 自动显示章标题** (复选框) — 选中后页眉将自动显示当前章节标题。
- **偶数页对齐** (下拉菜单) — 设置偶数页页眉的对齐方式。
- **页眉字体** (输入框, 默认: "宋体") — 设置页眉文本的字体。
- **页眉字号** (下拉菜单, 默认: 10.5pt) — 设置页眉文本的字号。
- **页眉文字加粗** (复选框, 默认: 未选中) — 控制页眉文本是否加粗。
- **页眉下划线** (复选框, 默认: 选中) — 控制页眉下方是否显示分隔线。
- **下划线粗细** (单位: 磅, 默认: 0.75) — 设置页眉下划线的粗细。
- **下划线样式** (下拉菜单) — 选项："单线"、"双线"。

### 页码

- **前置页码位置** (下拉菜单, 默认: "居中")
  - 设置前置页（如摘要、目录）页码的显示位置。选项：居中 / 居左 / 居右。
- **正文页码位置** (下拉菜单, 默认: "居中")
  - 设置正文页码的显示位置。选项：居中 / 居左 / 居右 / 奇右偶左。
- **前置页码格式** (下拉菜单)
  - 选项：大写罗马 (I, II, III) / 小写罗马 (i, ii, iii) / 阿拉伯数字 (1, 2, 3)。
- **正文页码格式** (下拉菜单) — 同上。
- **前置起始编号** (默认: 1) — 设置前置页码的起始数字。
- **正文起始编号** (默认: 1) — 设置正文页码的起始数字。
- **页码修饰** (输入框, 默认: "{page}", 提示: 如 - {page} -)
  - 在页码周围添加修饰文本。`{page}` 是页码占位符。例如输入 `- {page} -` 会显示为 "- 1 -"。
- **页码字体** (输入框, 提示: 空=跟随西文) — 设置页码字体。留空则跟随西文字体。
- **页码字号** (下拉菜单, 默认: 10.5pt) — 设置页码字号。
- **页码加粗** (复选框, 默认: 未选中) — 控制页码是否加粗。

## 选项卡 5: 目录与参考文献

此选项卡用于设置目录样式、参考文献缩进以及特殊标题的映射规则。

### 目录

- **目录深度** (单位: 级, 默认: 3) — 控制目录中显示的标题层级。设置为 3 级则显示一至三级标题。
- **二级条目字体** (输入框, 默认: "宋体") — 设置目录中二级及以下条目的字体。
- **二级条目字号** (下拉菜单, 默认: 12pt) — 设置目录中二级及以下条目的字号。
- **一级条目字体** (输入框, 默认: "黑体") — 设置目录中一级标题条目的字体。
- **一级条目字号** (下拉菜单, 默认: 14pt) — 设置目录中一级标题条目的字号。
- **目录行距** (单位: 倍, 默认: 1.5) — 设置目录条目的行间距。
- **条目段前** (单位: 行, 默认: 0) — 设置目录条目上方间距。
- **条目段后** (单位: 行, 默认: 0) — 设置目录条目下方间距。

### 参考文献

- **参考文献缩进** (单位: pt, 默认: 24) — 设置参考文献条目的悬挂缩进量。24pt 相当于两个汉字的宽度。

### 特殊标题映射

此区域用于定义特殊标题（如摘要、目录、参考文献、致谢）在最终文档中的显示方式。

表格包含三列：
- **匹配** — 原始文档中标题的文本。
- **显示** — 希望在最终文档中显示的样式（例如加大字间距）。
- **对齐** — 标题的对齐方式。

默认映射：

| 匹配 | 显示 | 对齐 |
|------|------|------|
| 摘要 | 摘&emsp;&emsp;&emsp;&emsp;要 | 居中 |
| 目录 | 目&emsp;&emsp;&emsp;&emsp;录 | 居中 |
| 参考文献 | 参&ensp;考&ensp;文&ensp;献 | 居中 |
| 致谢 | 致&emsp;&emsp;&emsp;&emsp;谢 | 居中 |

点击 **"添加"** 按钮可新增一行映射规则，点击 **"删除末行"** 可删除最后一行。

## 选项卡 6: 图表与三线表

此选项卡用于设置图表标题样式、编号检查以及三线表的线条样式。

- **图表题字号** (下拉菜单, 默认: 10.5pt) — 设置所有图片标题和表格标题的字号。
- **图表题防分页** (复选框, 默认: 选中) — 建议保持选中。确保图表标题与其内容不会被分页分隔。
- **检查图表编号连续性** (复选框, 默认: 选中) — 建议保持选中。工具会检查图表编号是否连续。

---

- **图题模式** (输入框, 默认: `^图\s*\d`, 提示: 如 图1) — 定义识别图片标题的模式。
- **表题模式** (输入框, 默认: `^(续)?表\s*\d`, 提示: 如 表1) — 定义识别表格标题的模式。
- **分图模式** (输入框, 默认: `^\([a-z]\)`, 提示: 如 (a)) — 定义识别子图标题的模式。
- **表注模式** (输入框, 默认: `^注[：:]`, 提示: 如 注：) — 定义识别表格注释的模式。

---

### 三线表

- **顶线粗细** (单位: 磅, 默认: 1.5) — 三线表顶部横线的粗细。
- **栏目线粗细** (单位: 磅, 默认: 1.0) — 三线表栏目行下方横线的粗细（中间那条线）。
- **底线粗细** (单位: 磅, 默认: 1.5) — 三线表底部横线的粗细。
- **表格行距** (单位: 倍, 默认: 1.0) — 三线表内部文字的行间距。

---

## 选项卡 7: 封面与声明页

此选项卡用于设置论文封面内容和声明页文本。

### 封面

- **启用封面** (复选框, 默认: 选中) — 控制是否由工具生成封面页。选中后下方设置才生效。
- **自定义封面** (输入框 + 浏览按钮) — 如果您已有排好版的 `.docx` 封面文件，可通过"浏览"按钮上传，工具将使用该封面替代自动生成的封面。

---

- **学校名称** (输入框, 默认: "华南农业大学") — 封面页显示的学校名称。
- **Logo 文件** (输入框 + 浏览按钮, 默认: "scau_logo.png") — 封面页的学校 Logo 图片。
- **封面标题** (输入框, 默认: "本科毕业论文(或设计)") — 封面页的论文类型标题。
- **信息栏字段** (动态行) — 定义封面上的学院、专业、姓名、学号、指导教师等信息字段。每行包含字段标签和下划线宽度。
  - 点击 **"添加"** 按钮新增字段，点击 **"删除末行"** 删除最后一个字段。

### 声明

- **启用声明页** (复选框, 默认: 选中) — 控制是否生成原创性声明和使用授权声明页。
- **声明 1: 原创性声明** — 可编辑标题和正文内容，确保符合学校要求。
- **声明 2: 使用授权声明** — 可编辑标题和正文内容，确保符合学校要求。

---

## 底部操作栏

无论当前在哪个选项卡，界面底部的操作栏始终可见。

### 配置管理

- **"默认(SCAU)"** 按钮 — 将所有设置恢复到华南农业大学 2024 年本科毕业论文的默认格式。
- **"保存"** 按钮 — 将当前所有设置保存为 YAML 配置文件，方便下次使用或分享给同学。
- **"加载"** 按钮 — 从之前保存的配置文件中加载设置。

### 文件选择

- **输入** — 选择要格式化的论文原稿。点击右侧"浏览"按钮打开文件选择对话框。
- **输出** — 指定格式化后文件的保存路径。默认会在原文件名后加 `_formatted` 后缀。

### 执行

- **跳过目录生成** (复选框) — 选中后将跳过自动目录生成步骤。目录生成需要系统已安装 Microsoft Word。
- **"开始格式化"** 按钮 — 确认所有设置后，点击此按钮开始格式化。
- **日志窗口** — 显示格式化过程中的实时日志、进度信息和可能的警告或错误。

---

## 支持的输入格式

| 格式 | 说明 | 额外依赖 |
|------|------|----------|
| `.docx` | Word 文档 | 无 |
| `.doc` | 旧版 Word 文档 | 需本机安装 Microsoft Word |
| `.txt` | 纯文本 | 需安装 [pandoc](https://pandoc.org/) |
| `.md` | Markdown | 需安装 pandoc |
| `.tex` | LaTeX | 需安装 pandoc |

输出统一为 `.docx` 格式。

---

## 重要注意事项

1. **`.docx` 输入**：标题的编号与标题文字之间需要有空格（如 `2.1 数字劳动`），无空格写法（`2.1数字劳动`）暂不支持。
2. **`.docx` 输入**：标题行必须顶格书写，行首不能有空格，否则无法识别。
3. **`.txt` 输入**：工具会根据文本模式自动检测标题，无需手动设置标题样式。
4. **目录生成**：自动目录生成功能需要系统已安装 **Microsoft Word**。如未安装，请勾选"跳过目录生成"。
5. **默认配置**：工具默认配置针对 **华南农业大学 2024 年本科毕业论文** 格式要求。其他学校的同学请务必根据自己学校的要求调整各项设置。
6. **封面、摘要、声明、参考文献**：当前不同学校对这些部分的格式要求差异较大，本工具不做统一排版处理，请根据学校要求自行调整。
7. **务必人工核对**：自动排版工具识别一定会有遗漏的地方，自动工具只是把比较麻烦的部分处理了，**请务必亲自核对最终文档**，部分细节仍需手动微调。
8. **脚注**：`.txt` 文件不支持脚注显示；`.docx` 文件能保留已有的脚注但不能自动插入新脚注。
9. **图表区域**：图片和表格处的文字排版没有做统一处理，需要人工自行调整。
10. **不会填选项？** 将你的 YAML 配置文件和学校格式要求文档发给 AI（如 ChatGPT、Claude），让 AI 教你工具里看起来复杂的正则表达式或选项应该怎么填。
